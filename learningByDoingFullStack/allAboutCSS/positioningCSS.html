Sifat-sifat positining dalam property position CSS

  • Ada beberapa sifat dari positioning yang dibagi jadi dua bagian umum, yakni static flow dan non-static flow. Perbedaannya lebih mudah dipahami lewat view 3d yang dijelaskan pada modul dasar pengembangan web by dicoding - perbedaan static flow vs non-static flow

    • Ini kombinasi dari relative + fixed. Relative = dia awalnya terletak disini. Fixed = dia bakalan nggantung ditop kalau discroll ke bawah.

    Static flow/normal flow itu sifat positioning pada default elemen block yang selalu berada dibaris baru/dibawahnya ketika dibuat. Jadi, setiap elemen block selalu muncul di bawah dari elemen block sebelumnya. Bahkan, jika masih terdapat ruang kosong pada samping elemennya, mereka tidak akan tampak bersebelahan.
    Nih contohnya, dia dibatasi kan ukuranya jadi doang, tapi kalau diperbesar lagi sebetulnya kan masih muat paragraph dibawahnya berada disampingnya.

    Relative positoing itu seperti membuat elemen dapat melakukan perpindahan posisi ke atas, kanan, bawah, ataupun kiri dari posisi semula atau posisi seharusnya elemen tersebut berada. Perpindahan posisi ini tidak akan berpengaruh terhadap posisi elemen di sekitarnya karena ketika menggunakan relative positioning elemen tersebut akan dipindahkan dari normal flow.

    Absolute itu sama seperti relative, elemen akan dipindahkan keluar dari normal flow sehingga kita dapat memindahkan posisi elemen ke atas, kanan, bawah, ataupun kiri secara leluasa tanpa mengganggu elemen di sekitarnya. Namun, posisinya relatif terhadap jendela browser dan posisinya dapat relatif pada induk elemen selama induk elemen juga berada di luar dari normal flow. Untuk memahami relatifnya absolute terhadap browser/viewport tuh:
    Jadi nanti elemen yang berada pada normal flow/tanpa postion dan tanpa parent alias parentnya adalah viewport browser, elemen tersebut akan mengisi bagian atas/posisi static flow/normal flow elemen absolute karena dia diabaikan oleh elemen yang berada didalam normal flow.

    Ini contoh kalau induk element berada di luar normal flow alias diberi postion kayak relative misal! dia akan static berdasarkan ukuran top/bottom yang dia define terhadap ukuran lebar/tinggi si induk elemen.

    Ini contoh bila tetap berada di dalam normal flow si elemen induknya. Diabaikanlah dia ama si absolute positon alias tidak static.

    Ia merupakan absolute position. Namun, posisinya selalu relatif pada jendela browser, bahkan ketika pengguna melakukan scrolling posisinya akan tetap tampak pada posisinya di layar.
